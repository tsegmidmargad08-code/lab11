# Лабораторын ажил №11 — Таниулдулга Веб

## Тайлбар

Front-end portfolio website with admin panel — GitHub Pages дээр ажиллах боломжтой.

## Файлын бүтэц

```
lab11/
├── index.html          ← Үндсэн portfolio хуудас
├── css/
│   ├── style.css       ← Portfolio stylesheet
│   └── admin.css       ← Admin panel stylesheet
├── js/
│   └── main.js         ← Portfolio JS
└── admin/
    ├── index.html      ← Нэвтрэх хуудас (/admin/index.html)
    ├── dashboard.html  ← Админ удирдлага
    ├── menu.html       ← Цэс удирдах
    └── news.html       ← Мэдээ удирдах
```

## Хэрхэн ашиглах

### Portfolio (index.html)
- Navbar: Миний үр чадвар / Бүтээсэн төсөл / Холбоо барих
- `[Таны нэр]` гэсэн хэсгийг өөрийн нэрээр солино

### Админ хэсэг (/admin/index.html)
- **Нэвтрэх нэр:** `admin`
- **Нууц үг:** `admin123`
- Цэс нэмэх, устгах
- Мэдээ нэмэх, устгах

## GitHub Pages дээр байрлуулах

1. Repository үүсгэх (жишээ: `lab11`)
2. Файлуудыг push хийх
3. Settings → Pages → Source: `main` branch → Save
4. URL: `https://[username].github.io/lab11/`
5. Админ: `https://[username].github.io/lab11/admin/`
